# 达人探店项目
## 项目介绍
是一个前后端分离项目，类似于大众点评，实现了发布查看商家，达人探店，点赞，关注等功能，业务可以帮助商家引流，增加曝光度，也可以为用户提供查看提供附近消费场所，主要用来配合学习Redis的知识
## 技术栈
SpringBoot+MySql+Lombok+MyBatis-Plus+Hutool+Redis
